Should you play golf?

A demo MLOps project that uses a decision tree classifier to decide if you should play golf.
Made by: github.com/pugad
Available APIs:
  • Train the model
    • Method: POST /train
    • Payload: JSON array
      • outlook (string):
        "sunny" or "overcast" or "rainy"
      • temperature (int; deg Celsius):
        e.g. 30
      • humidity (float):
        e.g. 0.10
      • windy (bool):
        true or false
      • play (string):
        "yes" or "no"
    • Example:
      [{"outlook": "sunny", "temperature": 26, "humidity": 0.186, "windy": true, "play": "yes"}, ...]
  • Request a prediction (should you play golf, given the submitted inputs?)
    • Method: POST /predict
    • Payload: JSON
      • outlook (string):
        "sunny" or "overcast" or "rainy"
      • temperature (int; deg Celsius):
        e.g. 30
      • humidity (float):
        e.g. 0.10
      • windy (bool):
        true or false
    • Example:
      {"outlook": "sunny", "temperature": 20, "humidity": 0.6, "windy": false}
  • Health check
    • Method: GET /healthz
